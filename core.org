This should ideally be the absolute bare minimum required to bootstrap an emacs
you can work with.

* Enable lexical binding
#+begin_src emacs-lisp
  ;; -*- lexical-binding: t -*-
#+end_src

* Use [[https://github.com/hrs/sensible-defaults.el][sensible-defaults]]
  #+begin_src emacs-lisp
    (load-file (concat user-emacs-directory "sensible-defaults.el"))
    (sensible-defaults/use-all-settings)
    (sensible-defaults/use-all-keybindings)
    (sensible-defaults/backup-to-temp-directory)
  #+end_src

* Text-wrapping
** Wrap paragraphs (fill) automatically
    Org mode inherits from text mode, so both will benefit from the following:

    #+begin_src emacs-lisp
      (add-hook 'text-mode-hook 'auto-fill-mode)
    #+end_src

** [[https://www.emacswiki.org/emacs/TruncateLines][Truncate lines]] (i.e. don't soft-wrap) by default
    #+begin_src emacs-lisp
    (set-default 'truncate-lines t)
    #+end_src

* Favour unix line-endings
  #+begin_src emacs-lisp
    (set-buffer-file-coding-system 'unix 't)
  #+end_src

* Useful functions
** Kill other buffers (like Bufonly)
   #+begin_src emacs-lisp
     (defun kill-other-buffers ()
       "Kill all other buffers."
       (interactive)
       (mapc 'kill-buffer
             (delq (current-buffer)
                   (org-remove-if-not 'buffer-file-name (buffer-list)))))
   #+end_src

** Provide function for fixing interaction between (minor) mode and evil
   #+begin_src emacs-lisp
     (defun my/keymap-fix-with-evil-mode (mode &optional hook map)
       "Allows (minor) mode to use evil as a base, but its keybindings take precedence.

     Solution derived from: https://github.com/emacs-evil/evil/issues/511
     "
       (let* ((base (symbol-name mode))
	      (hook (if hook hook (intern (concat base "-hook"))))
	      (map (if map map (intern (concat base "-map")))))

	 (evil-set-initial-state mode 'normal)
	 (eval-after-load mode
	   '(progn
	      ;; module's keybindings should have stronger priority than
	      ;; normal-mode keybindings:
	      (evil-make-overriding-map map 'normal)
	      ;; force evil keymaps to update after module loaded:
	      (add-hook hook #'evil-normalize-keymaps)))))
   #+end_src
